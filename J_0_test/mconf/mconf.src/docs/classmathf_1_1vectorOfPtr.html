Detailed Description

template<typename pT>
class mathf::vectorOfPtr< pT >

The class provides vector of pointers to simplify dealocation.

pT is the pointer to some object, memory for which was allocated by new The memory is deallocated by clear() or automatically by destructor when this class is deleted.
